# Home Feature

## Purpose

This feature owns the top-level home page composition.

## What Lives Here

- `components`: The page shell, marketing content, and the session-aware access panel wrapper.

## How It Fits Into The System

The `home` feature composes the other major frontend domains:

- Marketing content explains the stack.
- The session feature supplies state.
- The auth or account feature renders depending on session state.
